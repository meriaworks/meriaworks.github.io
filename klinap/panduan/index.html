01Beli & terima email

Semuanya otomatis — tidak perlu chat siapa pun.

  1. Klik Beli Sekarang di halaman harga, lalu bayar (QRIS / transfer / e-wallet).
  2. Dalam hitungan detik, cek email Anda: berisi KODE LISENSI dan tautan unduh paket.
  3. Simpan email itu baik-baik — kode lisensi Anda dipakai lagi bila suatu saat ganti komputer.
Email tidak masuk? Cek folder Spam/Promosi. Masih tidak ada setelah 15 menit → hubungi meriaworks.id@gmail.com dengan bukti pembayaran (ini satu-satunya skenario yang butuh email).

02Unduh & pasang installer

Lakukan di 1 komputer yang akan menjadi "pusat" klinik (komputer yang paling sering menyala).

  1. Unduh KlinapSetup.exe dari tautan di email, lalu klik dua kali untuk menjalankan.
  2. Bila muncul layar biru "Windows protected your PC" (SmartScreen): klik More info → Run anyway. Ini normal untuk aplikasi baru yang belum populer — bukan tanda bahaya.
  3. Ikuti installer: Next → Next → Install. Proses ±5–10 menit (installer sudah membawa semua yang dibutuhkan).
  4. Selesai → muncul ikon Klinap di Desktop. Klik dua kali untuk membuka.
Antivirus menahan file? Pilih "Allow/Izinkan" atau tambahkan folder C:\Klinap ke pengecualian (exclusion) antivirus. Klinap 100% offline dan tidak mengirim data ke mana pun.

03Aktivasi lisensi

Sekali saja, langsung dari email pembelian.

  1. Buka email pembelian → salin seluruh kode lisensi (klik 3× pada kodenya untuk memblok semua; boleh ikut terbawa spasi, sistem membersihkannya).
  2. Centang persetujuan Ketentuan Penggunaan (EULA) — bisa dibaca dulu lewat tautannya.
  3. Tempel kode → klik Setuju & Aktifkan Klinap. Pastikan internet masih tersambung saat langkah ini saja (verifikasi pembelian satu kali).
  4. Muncul halaman login = aktivasi berhasil. Selamanya aktif, 100% offline setelah ini.
"Kode lisensi tidak valid"? Hampir selalu karena kode tersalin sebagian — salin ulang seluruh baris dari email, lalu tempel lagi. Bila pesan menyebut internet: sambungkan internet sebentar dan klik Aktifkan lagi (hanya perlu sekali ini).

04Login pertama & amankan password

  1. Masuk dengan superadmin / ClinicOS#2026.
  2. WAJIB: buka User Management → ganti password superadmin dengan milik Anda sendiri.
  3. Buat akun untuk tiap peran staf: Front Desk (registrasi+kasir), Dokter, Perawat, Apoteker, dan Owner (untuk Anda: laporan & pengaturan). Tiap peran otomatis hanya melihat menu yang relevan.
  4. Staf login memakai akunnya masing-masing — semua aktivitas tercatat atas nama masing-masing di audit log.
Lupa password suatu saat? Tidak masalah — lihat bagian Mitigasi: ada tombol pemulihan Reset-Admin bawaan. Anda tidak akan pernah terkunci dari data sendiri.

05Setup awal klinik (±10 menit)

  1. Pengaturan Klinik → isi nama klinik, alamat, telepon, nomor izin (muncul di kop surat & struk).
  2. Pengaturan → Modul Klinik → centang jenis klinik Anda: Umum / Gigi / Estetik / Obgyn-KIA (boleh lebih dari satu). Fitur khususnya menyesuaikan otomatis.
  3. Kasir → tab Tarif → masukkan tarif jasa Anda (mis. "Konsultasi Dokter Umum — 50.000", "Cabut Gigi — 150.000").
  4. Apotek → tambah obat (atau Import dari Excel memakai template yang disediakan — tidak perlu ketik ulang), isi harga jual per obat, lalu catat stok masuk per batch beserta tanggal kedaluwarsanya.
  5. Pasien lama? Pasien → Import dari Excel/CSV memakai template.
Tidak wajib sekaligus. Minimal untuk mulai beroperasi: nama klinik + 1 tarif konsultasi + akun staf. Obat dan pasien bisa dicicil sambil jalan.

06Alur kerja harian

Satu pasien, dari datang sampai pulang — semua tersambung otomatis:

1 · Registrasi→2 · Antrean→3 · Periksa (RME)→4 · Resep → Apotek→5 · Kasir
  1. Front desk: menu Registrasi → pilih/daftarkan pasien → pilih poli → pasien otomatis dapat nomor antrean. (Layar antrean untuk TV ruang tunggu: menu Antrean → Layar Panggil.)
  2. Dokter: menu RME/SOAP → pilih pasien dari antrean → tulis S-O-A-P + diagnosis ICD-10 (ketik untuk mencari) → buat e-resep → Kirim ke Apotek. Rekam medis terkunci setelah final (hanya bisa ditambah addendum — sesuai kaidah).
  3. Apoteker: menu Resep → resep masuk otomatis → siapkan → Dispensing: stok terpotong otomatis dengan FEFO (kedaluwarsa terdekat keluar duluan).
  4. Kasir: menu Kasir → kunjungan muncul di "Belum Ditagih" → item obat sudah terisi dari resep → tambah jasa dari tarif → Terima Pembayaran → struk tercetak.

07Kasir & struk

  1. Tarif (sekali saja): Kasir → tab Tarif → tambah semua jasa/tindakan klinik Anda.
  2. Menagih: tab Kasir → klik kunjungan di "Belum Ditagih" → item obat otomatis terisi dari resep (harga dari data obat) → tambah jasa dari dropdown tarif → isi diskon bila ada.
  3. Menerima pembayaran: klik Terima Pembayaran → pilih Tunai/QRIS/Transfer/Debit → untuk tunai, ketik uang diterima → kembalian dihitung otomatis → struk tercetak.
  4. Salah input? Tagihan bisa dibatalkan dengan alasan (tercatat permanen di audit — aman dari kecurangan).
  5. Rekap: ringkasan pendapatan hari ini selalu tampil di atas halaman Kasir; riwayat per tanggal ada di tab Riwayat.
Printer struk: printer kasir 58/80mm apa pun yang sudah terpasang di Windows langsung bisa dipakai (dialog cetak browser). Printer biasa (A4) juga bisa.

08Banyak komputer dalam satu klinik (LAN)

1 lisensi = 1 klinik, bebas jumlah perangkat. PC lain tidak perlu diinstal apa pun.

  1. Pastikan semua perangkat tersambung ke Wi-Fi/jaringan yang sama dengan PC utama.
  2. Di PC utama, lihat alamat yang tertera saat Klinap menyala (contoh: http://KOMPUTER-KLINIK:3001 atau http://192.168.1.10:3001).
  3. Di PC/tablet/HP lain: buka browser (Chrome/Edge) → ketik alamat itu → login dengan akun masing-masing. Selesai.
  4. Simpan sebagai pintasan: buka menu browser → Add to Home screen / Install app.
PC lain tidak bisa membuka? 99% karena firewall. Di PC utama: saat pertama kali Klinap jalan, Windows menanyakan izin jaringan — pilih Allow/Izinkan (Private network). Terlanjur menolak? Buka Windows Security → Firewall → Allow an app → centang Klinap/Node.js untuk Private.

09Backup & pemulihan

  1. Otomatis: Klinap membuat backup setiap hari sendiri, disimpan 30 salinan terakhir. Tidak perlu melakukan apa pun.
  2. Kebiasaan baik (1×/minggu, 1 menit): menu Backup & Restore → Unduh backup → simpan ke flashdisk/Google Drive. Ini melindungi dari skenario terburuk: komputer hilang/mati total.
  3. Memulihkan: menu Backup → pilih file → Restore. Sistem memeriksa keutuhan file dulu, dan menyimpan salinan pengaman sebelum memulihkan — tidak bisa "tambah rusak".
Untuk yang teliti: lokasi backup bisa dipindah ke disk kedua/USB permanen (variabel BACKUP_DIR) — sehingga kerusakan disk utama tidak ikut memakan backup.

🛟Mitigasi — bila hal buruk terjadi

Semua skenario di bawah bisa Anda selesaikan sendiri, tanpa menghubungi siapa pun.

MasalahSolusi mandiri
Lupa password / semua akun terkunciDi PC utama, klik dua kali Reset-Admin.bat (di folder Klinap / Start Menu) → password superadmin kembali ke ResetKlinap#2026 → masuk → ganti password. Selalu berhasil, offline.
Internet matiBukan masalah — Klinap memang tidak butuh internet. Semua fitur tetap jalan.
Listrik padam saat sedang inputDatabase Klinap tahan padam (mode jurnal aman). Nyalakan lagi → data terakhir yang tersimpan tetap utuh.
Komputer rusak / hilang / ganti PC baruDi PC baru: pasang installer (unduh lagi dari email) → aktivasi dengan kode lisensi yang sama → menu Backup → Restore file backup terakhir Anda. Semua data kembali. (Karena itu, rajinlah menyalin backup ke flashdisk.)
Kode lisensi hilangCari email pembelian Anda (cari kata "KLINAP2" di kotak masuk). Kode tidak pernah kedaluwarsa.
Salah hapus obat / pasien / BHPTidak ada penghapusan permanen — semuanya diarsipkan. Buka daftar → filter Arsip → Pulihkan.
Salah input tagihan yang sudah lunasKasir → buka tagihan → Batalkan dengan alasan → buat tagihan baru. Semua tercatat di audit log.
Antivirus/SmartScreen memblokir installerSmartScreen: More info → Run anyway. Antivirus: Allow, atau tambahkan folder C:\Klinap ke exclusion. Klinap tidak mengirim data ke mana pun.
PC lain di klinik tidak bisa membuka KlinapPastikan satu jaringan Wi-Fi → izinkan Klinap/Node di Windows Firewall (Private) → coba pakai alamat IP, bukan nama komputer.
Aplikasi tidak mau terbukaRestart komputer dulu (menyelesaikan 90% kasus). Masih gagal: klik kanan ikon Klinap → Run as administrator. Data Anda aman — tersimpan terpisah dari aplikasi.
Layar menampilkan error saat dipakaiMuat ulang halaman (F5). Kejadian berulang di menu yang sama → catat tulisannya, lanjutkan kerja dari menu lain; data yang sudah tersimpan tidak hilang.
Staf nakal mengubah/menghapus dataSemua aksi tercatat di Audit Log (siapa, kapan, dari komputer mana, sebelum-sesudah) dan tidak bisa dihapus. Batasi hak akses per peran di User Management.

?FAQ singkat

  • Berapa lama sampai bisa dipakai? ±15 menit dari bayar sampai registrasi pasien pertama.
  • Perlu langganan? Tidak. Sekali bayar Rp 129.000, selamanya.
  • Perlu internet? Hanya saat mengunduh installer. Setelah itu 100% offline.
  • Data saya di mana? Di komputer Anda sendiri. Tidak ada server pihak ketiga.
  • Boleh dipakai di 2 klinik? 1 lisensi untuk 1 klinik (banyak PC dalam 1 jaringan boleh). Klinik kedua → beli lisensi kedua.
  • Masih ada pertanyaan? Tanya asisten otomatis di beranda (pojok kiri bawah), atau baca ulang bagian terkait di atas.
